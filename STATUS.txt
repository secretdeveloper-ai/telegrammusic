QUICK REFERENCE:
================================================================================

Music Commands:
  /play <song>          Play a song
  /skip                 Skip current song
  /next                 Next in queue
  /queue                Show queue
  /shuffle              Random order
  /remove <pos>         Remove song
  /clear_queue          Empty queue

Admin Commands:
  /init                 Setup group
  /admin_add <id>       Make admin
  /admin_remove <id>    Remove admin
  /ban <id>             Block user
  /unban <id>           Unblock user
  /set_prefix !         Change prefix
  /queue_limit 50       Max songs

Owner Only:
  /broadcast <msg>      Announce to ALL groups

Other:
  /start                Start bot
  /help                 All commands
  /ask <question>       Ask ChatGPT
  /stats                Statistics
  /about                About bot

## 🚀 QUICK START

### Step 1: Get Telegram ID
```
1. Open Telegram
2. Search: @userinfobot
3. Message: /start
4. Copy your ID
```

### Step 2: Upload to GitHub
```
1. Go to github.com
2. New repository
3. Follow GITHUB_UPLOAD_GUIDE.md
4. Upload all files
```

### Step 3: Deploy on Railway
```
1. Go to railway.app
2. New Project → Deploy from GitHub
3. Add variables (from .env.example)
4. Change OWNER_ID to your ID
5. Deploy!
6. Wait 3 minutes
7. Done! ✅
```
